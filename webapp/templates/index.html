WhatsApp

Patcher
Signature Bypass View-Once Save Anti-Delete Firebase Restore
v2.26.7.70 (260707004)

Other versions may not work — protobuf field names (viewOnce_, key_) and method signatures change across versions. Always test first.

Features

  • View-Once — flips viewOnce_=false on the decrypted protobuf, WhatsApp saves the media normally
  • Anti-Delete — invalidates delete messages by replacing the message ID with a dummy
  • Firebase Restore — re-signing the APK breaks Google Maps inside WhatsApp because the API key is tied to the original certificate. This patch extracts the original google_api_key from resources and injects it back at runtime, so location sharing and map previews keep working. Optional: you can also provide your own API key with Maps SDK + Places API enabled

Known Issue

  • Google Drive backup broken — patched APK has a different signature, so Google Drive rejects it (#69). A fix is available on the experimental branch (GMS bypass for Drive backup without Play Services) but still needs testing.
